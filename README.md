## Project Description
This repository contains automated test cases developed using **Playwright** to test the **Singlish to Sinhala conversion** functionality of the web application available at:

https://www.swifttranslator.com/

## Prerequisites

- Node.js (LTS version)
- Git
- Visual Studio Code

---

## Installation and Execution Instructions

Follow the steps below to install dependencies and run the tests:

```bash
# 1. Clone the repository
git clone https://github.com/AlokaSenevirathne/IT23667600-playwright-automation.git

# 2. Navigate to the project directory
cd IT23667600-playwright-automation

# 3. Install project dependencies
npm install

# 4. Run all Playwright tests
npx playwright test

# 5. View the HTML test execution report
npx playwright show-report
